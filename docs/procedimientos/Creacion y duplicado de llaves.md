# Creacion de llaves

**Categoría:** Recepcion
**Última actualización:** 22/06/2026
**Validado por:** Martin

### Resumen rápido

- Abrir la plataforma de llaves visionline e introducir usuario (rec) y contraseña (rec)
- Colocar la llave magnetica en el lector de llaves
- Rellenar los campos en el generador. Numero de habitacion, cantidad de noches y cantidad de llaves.
- Seleccionar make key y esperar a los tres pitidos.

### Procedimiento detallado

1. Encontramos el icono en el escritorio. Suele estar ejecutandose en todo momento.
2. El lector de llaves es el dispositivo negro al lado del teclado
3. Si es una llegada seleccionamos Guest, pero si fuera una copia debemos seleccionar joiner.
4. Se selecciona override excepto que sea una llave de piscina o salones.

### Casos especiales / excepciones
- Si queremos llaves de salones, debemos ticar el nombre del salon.
- Si fuera llave de salon o piscina, en numero de habitacion ponemos 1000
- Siempre pedir documentacion para duplicar una llave.
