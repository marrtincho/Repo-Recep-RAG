# Creacion de reserva

**Categoría:** Reservas
**Última actualización:** 22/06/2026
**Validado por:** Martin

### Resumen rápido

- Para comrobar la tarifa lo miramos en el sitio web del hotel.
- Damos a reservas > Reserva > Nueva reserva (Alt +R)
- Selecionamos primero el tipo de tarifa (BAR), luego la tarifa y el nombre del cliente.
- Rellenamos el resto de campos: cantidad de personas, tipo de habitacion, pension y fecha de llegada y salida.

### Procedimiento detallado

1. Colocamos fechas y cantidad de noches, y nos saldra la tarifa por tipo de habitacion.
2. Los atajos nos agilizan las tareas.
3. Si no seleccionamos primero el tipo de tarifa, se borra. El precio es el que salga en la web o el que hayamos acordado con el cliente.
4. Segmento de reserva: Turismo independiente, origen de reserva: directo offline, canales de reserva: PMS.

### Casos especiales / excepciones
- Si tienen tarifa corporativa deberan enviar un correo a reservas.
- Si tienen mejor tarifa online deberan hacer la reserva por la plataforma online.

### Cuándo escalar
- Siempre prestar atencion al comportamiento del cliente para ahorarnos un posible problema.
- Siempre cobrar al check in y tomar deposito en efectivo.
