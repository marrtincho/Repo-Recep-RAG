# Creación de reserva

**Categoría:** Reservas
**Última actualización:** 01/01/2025
**Validado por:** Carlos López

### Resumen rápido

- Para comprobar la tarifa, consultarla en la web oficial del hotel o en la tabla de tarifas vigente.
- Ir a Reservas > Reserva > Nueva reserva (atajo: Alt+R).
- Seleccionar primero el tipo de tarifa (BAR u otro), luego la tarifa concreta y el nombre del cliente.
- Rellenar el resto de campos: número de personas, tipo de habitación, régimen y fechas de entrada y salida.

### Procedimiento detallado

1. Introducir las fechas y el número de noches; el sistema mostrará la tarifa disponible por tipo de habitación.
2. Los atajos de teclado agilizan el proceso; consultar el manual de sistema para la lista completa.
3. Seleccionar el tipo de tarifa antes que la tarifa concreta; de lo contrario, el sistema borra la selección. El precio debe coincidir con el publicado en la web o con el acordado directamente con el cliente.
4. Segmento de reserva: Turismo independiente. Origen: directo offline. Canal: PMS.

### Casos especiales / excepciones

- Si el cliente tiene tarifa corporativa, debe enviar la solicitud al departamento de reservas por correo.
- Si el cliente encuentra una tarifa más económica en una plataforma online, debe gestionar la reserva por esa vía.

### Cuándo escalar

- Estar atento al comportamiento del cliente para anticipar posibles incidencias.
- Cobrar siempre al check-in y recoger la fianza correspondiente.
