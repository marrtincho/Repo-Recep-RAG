## Preguntas habituales de concepto

- "¿Qué concepto uso para el coffee con comida?" — Coffee break (banquetes), 75 % comida y 25 % bebida
- "¿Qué concepto uso para un cóctel?" — Cocktail (banquetes), 75 % comida y 25 % bebida
- "¿Qué concepto uso para el menú de gala?" — Banquete, 75 % comida y 25 % bebida
- "¿Qué concepto uso para el menú de trabajo?" — Almuerzo trabajo, 75 % comida y 25 % bebida
- "¿Qué concepto uso para un ramo de flores?" — Ventas externas al 10
- "¿Qué concepto uso para los audiovisuales?" — ALQUS si son externos, AUS si son del hotel
- "¿Qué concepto uso para la lavandería?" — Lavado de ropa
- "¿Qué concepto uso para el spa?" — Servicio Spa
- "¿Qué concepto uso para el maletero?" — Venta prestación de servicio

---

## Por qué importa mantener esto consistente

- Un mismo gasto cargado con conceptos distintos dificulta la conciliación contable a final de mes.
- Usar un concepto incorrecto puede aplicar un tratamiento de IVA inadecuado.
- Esta tabla es la referencia única para evitar interrumpir a un responsable por cada duda de concepto.

---

## Notas de mantenimiento

- Añadir una fila cada vez que surja un cargo que no encaje en ningún concepto existente.
- Revisar cuando se cree un concepto nuevo en el sistema o cambie la política de facturación.
- Revisión periódica recomendada: trimestral, o inmediatamente si administración comunica un cambio.
