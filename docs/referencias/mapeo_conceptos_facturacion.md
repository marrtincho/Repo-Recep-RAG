**Última revisión:** 25/06/2026
**Responsable de mantener actualizado:** Martin

*Documento de consulta directa. Evita que cada recepcionista elija el concepto "que más se parece" por intuición, lo cual genera inconsistencias contables difíciles de rastrear después. Si tienes dudas sobre un cargo que no está aquí, consulta con administración antes de cargarlo y luego añádelo a esta tabla.*

## Preguntas habituales de concepto

- "¿Qué concepto uso para el coffee español?" — Coffee break (banquetes), 75% comida y 25% bebida
- "¿Qué concepto uso para el finger ligero?" — Cocktail (banquetes), 75% comida y 25% bebida
- "¿Qué concepto uso para el menu chef?" — Banquete, 75% comida y 25% bebida
- "¿Qué concepto uso para el menu de trabajo?" — Almuerzo trabajo, 75% comida y 25% bebida
- "¿Qué concepto uso para un ramo de flores?" — Ventas externas al 10
- "¿Qué concepto uso para los audiovisuales?" — ALQUS si son externos, AUS si son del hotel
- "¿Qué concepto uso para la lavanderia?" — Lavado de ropa
- "¿Qué concepto uso para el SPA?" — Servicio Spa 10
- "¿Qué concepto uso para el maletero?" — Venta prestacion de servicio
- "¿Qué concepto uso para el coffee liquido?" — Coffee break bebida, 100% bebida

---

## Por qué importa mantener esto consistente

- Un mismo gasto cargado con conceptos distintos dificulta la conciliación contable a final de mes.
- Usar un concepto que no corresponde puede aplicar un tratamiento de IVA incorrecto.
- Esta tabla es la referencia única para evitar interrumpir a un responsable por cada duda de concepto.

---

## Notas de mantenimiento de este documento

- Añadir una fila cada vez que surja un cargo que no encajaba claramente en ningún concepto existente
- Revisar cuando se cree un concepto nuevo en el sistema o cambie la política de facturación
- Revisión periódica recomendada: trimestral, o de inmediato si administración comunica un cambio
